Principais formatações

Negrito ou Destaque

Nessa frase temos um termo em negrito usando a tag b (não semantica)

Já nesta frase, temos um termo em destaque usando a tag strong (semantica)

Italico ou Enfase

Nesta frase temos um termo em italico, usando a tag i (não semantica)

Já nesta frase temos um termo em Enfase, usando a tag em (semantica)

Texto marcado

Podemos criar tambem um texto marcado, usando a tag mark

Tambem temos outro mark com texto marcado neste paragrafo de exemplo

Texto excluido(del)

Aqui temos um texto como excluido, indicando que ele nao deve ser considerado, apesar de lido

Texto inserido (ins)

Aqui ja temos um texto inserido, indicando enfase e que foi adicionado depois

A tag U, é usada para sublinhar, parecendo ser da mesma maneira que a ins, porem não há semantica, apenas forma (css).

Texto sobrescrito (sup)

Para textos como X2 + 2, usamos a tag sup, para sobrescrever

Texto subscrito

Já para textos como CO2, H2O, e etc, usamos a tag sub, para subscritar
